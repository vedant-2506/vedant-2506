## 🚀 Projects

### 🌾 [FarmFeed](https://github.com/vedant-2506/FarmFeed)
An online fertilizer delivery platform connecting farmers with nearby shopkeepers. Farmers can browse fertilizer stock, place orders, and get quick supply — solving a real problem I noticed in rural supply chains. The project is deployed and live.

**Tech:** Java · HTML · CSS · JavaScript · Docker  
🔗 **Live:** [farm-feed-eosin.vercel.app](https://farm-feed-eosin.vercel.app)

---

### 🛒 [Fruitly](https://github.com/vedant-2506/Fruitly)
A complete e-commerce website for buying fruits online. Has user login, product listing, cart, and order management — all built from scratch using Java web stack. One of my first full end-to-end projects.

**Tech:** Java · JSP · Servlets · JDBC · MySQL · Bootstrap · CSS

---

### 🧮 [Numerical Computing](https://github.com/vedant-2506/Numericl_Computing)
C++ implementations of numerical methods from my MSc coursework. Covers root finding (bisection, Newton-Raphson), matrix operations, complex numbers, graph-based methods, and more. Useful reference for anyone studying scientific computing.

**Tech:** C++ · Gnuplot

---

### 💡 [LeetCode Solutions](https://github.com/vedant-2506/LeetCode_Solution)
My ongoing collection of LeetCode problem solutions. I focus on writing clean, readable code — not just solutions that pass. Mostly C++ right now.

**Tech:** C++
