**Tech used:** Java, JSP, Servlets, JDBC, MySQL, Bootstrap, CSS

### 🌾 [FarmFeed](https://github.com/vedant-2506/FarmFeed)
An online fertilizer delivery platform that connects farmers directly with shopkeepers. The idea came from a real gap I noticed — farmers often can't easily find fertilizer suppliers nearby. Built with Java backend + HTML/CSS/JS frontend, also has a Dockerfile for deployment. It's live on Vercel.

**Tech used:** Java, HTML, CSS, JavaScript, Dockerfile
